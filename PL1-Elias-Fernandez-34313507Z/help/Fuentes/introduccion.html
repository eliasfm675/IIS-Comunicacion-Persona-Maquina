Uniovi's pizzeria

Uniovi's pizzeria es una aplicacion que permite al usuario realizar una comanda para posteriormente consumir en el local. El usuario necesitará tener una reserva previa para hacer uso de la app.



La aplicacion consta de las siguientes secciones

  • Presentacion: una pantalla de bienvenida al usuario.
  • Datos: el usuario tendra que introducir su DNI/NIE y su codigo de reserva
  • Menu: se muestran los distintos productos que ofrece la pizzeria con sus tipos para el uso de filtros, además de un carrito para ver los items que llevas
  • Pago y revision: una pantalla donde se muestra un resumen del producto con opciones a cancelarlo, las observaciones del cliente, metodos de pago y la opcion de jugar o no
  • Juego(opcional): se muestra un juego de un tablero 4x4 en el cual si logras hacer pareja iras obteniendo descuentos
